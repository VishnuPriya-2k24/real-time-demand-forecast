# Real-Time Demand Forecasting System
ML-based demand forecasting using Azure ML and Power BI
## Overview
This project predicts product demand for an e-commerce platform using machine learning models and provides real-time insights through dashboards.
## Tech Stack
- Python
- Pandas, Scikit-learn, Statsmodels (ARIMA)
- Azure ML
- Power BI
## Features
- Demand prediction using time-series and regression
- Seller inventory optimization
- Customer demand analysis
- Real-time dashboard visualization
## Results
- Reduced stock-outs by ~25%
- Improved forecast accuracy by ~30%
